# Sistema de Turnos - Peluquería

Proyecto para la materia Programación 6 (Java - Cliente/Servidor).

## Tecnologías
- Java 21
- Spring Boot
- PostgreSQL
- HTML/CSS/JS (frontend)
- Git y GitHub

## Funcionalidades
- Login/registro de usuarios (dni + contraseña)
- Roles: Cliente (1), Barbero (2), Admin (3)
- Gestión de turnos
- Agenda por barbero
- Estados de turno: disponible, reservado, completado, cancelado
- Reportes para admin
